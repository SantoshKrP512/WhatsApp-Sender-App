**WhatsApp Bulk Sender Pro** is a high-performance, browser-based messaging platform designed for individuals and businesses to reach their contacts efficiently. Featuring a stunning glassmorphism UI, it transforms the simple task of messaging into a premium dashboard experience.

![Dashboard Preview](https://github.com/SantoshKrP512/WhatsApp-Sender-App/raw/main/dashboard_overview.png)

## ✨ Key Features

### 💎 Premium Experience
- **Glassmorphism UI**: A state-of-the-art dark theme with mesh gradients and smooth micro-animations.
- **Intuitive Navigation**: Sidebar-driven dashboard for seamless transitions between Single Send, Bulk Send, and Management tools.

### 📦 Bulk Messaging Engine
- **CSV Support**: Upload large contact lists in seconds using built-in high-speed parsing.
- **Dynamic Personalization**: Use `{name}` placeholders to automatically customize messages for each recipient.
- **Queue Management**: Real-time status tracking for every message in your bulk queue.
- **Smart Delay**: Configurable delays (1s-10s) to help manage delivery and respect platform limits.

### 🛠️ Pro Tools
- **Template Manager**: Save, delete, and reuse frequently sent messages.
- **Smart History**: Access your last 20 sent messages with timestamps.
- **Country Code Intelligence**: Automatic country code application with support for `+` prefixed international numbers.
- **Persistence**: All settings, templates, and history are persisted locally in your browser.

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Edge).
- (Optional) A local server environment like XAMPP if running from `htdocs`.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/SantoshKrP512/WhatsApp-Sender-App.git
   ```
2. Navigate to the directory:
   ```bash
   cd WhatsApp-Sender-App
   ```
3. Launch the app:
   Simply open `index.html` in your browser or visit your local server path (e.g., `http://localhost/WhatsApp-Sender-App/`).

## ⚙️ How to Use Bulk Sending
1. Prepare a `.csv` file with headers (e.g., `name, number`).
2. Go to the **Bulk Send** tab and upload your file.
3. Write your message. Use `{name}` where you want the contact's name to appear.
4. Set your preferred delay in **Settings**.
5. Click **Start Sending**. *Ensure popups are allowed for the site.*
